Temporary Draft: Data Dependency Structure

Purpose

This temporary page is for confirming the CO-LUMINATE data dependency structure before integrating into permanent documentation.

Core Data Inputs

Data asset Role in pipeline
Psychometric bridge source data Bridge source for SSQ/PHQ harmonization.
SSQ-only cohort response data Input scored with IRT-based calibration to align SSQ responses with PHQ-based depression reference logic.
Childhood exposure longitudinal data Source input for SDoH trajectory derivation in 0-5 and 6-12 year windows.
Integrated secondary analysis dataset Final integrated dataset joining calibrated depression outcomes and required covariates/exposures for secondary data analysis.

Dependency Flow (Draft)

Dependency Notes

  1. Psychometric bridge source data informs the calibration parameters.
  2. SSQ-only cohort response data is transformed using the calibration to make depression logic comparable.
  3. Childhood exposure longitudinal data contributes trajectory-derived social determinant features.
  4. Integrated secondary analysis dataset is the endpoint for CO-LUMINATE secondary analyses.

Counts in boxes are distinct participants (USUBJID) and are calculated at render time from available source objects. Integrated dataset count is restricted to participants with both SDoH trajectory coverage and a non-missing depression endpoint. Color mapping links workflow intent to project framing:

  • Violet nodes: proposal objectives.
  • Blue nodes: psychometric calibration stage.
  • Orange nodes: SDoH trajectory stage.
  • Green nodes: final linkage and integrated dataset stage.
