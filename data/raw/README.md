# Raw Data

This repository does not store raw study data.

## How To Obtain Raw Data

1. Confirm that you have the required AHRI approvals and data access permissions for the CO-LUMINATE project.
2. Request access through the approved AHRI data governance and custodianship process.
3. Retrieve the raw source data from the approved AHRI storage location after access is granted.
4. Keep raw data outside this public repository.
5. Use approved processing scripts to derive public-safe examples or release outputs before anything is promoted here.

## Rules

- Do not commit participant-level raw data to this repository.
- Do not record machine-specific absolute paths in committed files.
- Document provenance, version, and extraction date in release notes when public derivatives are produced.
