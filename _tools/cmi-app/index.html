Overview

This workspace helps the team capture, review, standardize, and export youth-informed causal chains. Start here for orientation, then move into the library, builder, and review workflow.

1. Orient Review the workflow and browser snapshot before editing or exporting any pathway.
2. Open From The Library Search, filter, and open an existing pathway by domain, polarity, wording, or narrative content.
3. Refine The Chain Adjust title, place, story note, node order, and review status while keeping the chain logic visible.
4. Review And Export Manage approval status, import prior work, and download structured outputs for analysis or handoff.

Library Tab

Use the Library tab to search by title, domain, node label, or story note, then open an approved or draft pathway for closer review.

Builder Tab

Use the Builder tab to work on one pathway at a time, keeping the node sequence, wording mode, and review status aligned.

  • Edit title, place, and story note
  • Add, remove, and reorder nodes
  • Save as draft, reviewed, or approved
