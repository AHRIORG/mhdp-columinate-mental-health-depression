Overview

This tab explains how the app is organized. Start here, then move into the library, builder, and review workflow.

1. Start Here Read the overview so you know what each tab is for before editing any pathway.
2. Find a Pathway Use the Library tab to search, filter, and open an existing pathway by domain or outcome type.
3. Build or Edit Use the Builder tab to adjust pathway wording, place, story note, node order, and outcome polarity.
4. Review or Export Use the Review and Export tab to manage status, import prior work, and download structured outputs.

Library Tab

Use the Library tab to search by title, domain, node label, or story note, then open a saved pathway.

  • Search pathways
  • Filter by poor or positive mental health
  • Browse domains and saved pathways

Builder Tab

Use the Builder tab to edit one pathway at a time, including its story note, place, node order, and outcome type.

  • Edit title, place, and story note
  • Add, remove, and reorder nodes
  • Save draft or move toward approval

Review & Export Tab

Use the Review & Export tab to manage review status, import prior work, and download structured outputs.

  • Review queue by status
  • Import JSON or CSV
  • Export JSON state, edge list, and summaries
